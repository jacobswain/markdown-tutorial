Image Links

Creating image links is similar to inline hyperlinks. The only difference is that image links begin with an exclamation mark ! 


            

                ![](https://www.google.com/images/srpr/logo11w.png)

You can add alt text to your images in the brackets before the URL. 


            

                ![Google Logo](https://www.google.com/images/srpr/logo11w.png)

Image links also support reference style syntax.

![alt text][reference name] 

                

                    ![Google][id]
                

                

                    [id]: https://www.google.com/images/srpr/logo11w.png "Image Title"

Example

Create a link to the Google logo at this URL: https://www.google.com/images/srpr/logo11w.png
