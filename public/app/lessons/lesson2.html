Headings

HTML supports headings using tags like h1 and h2 where h1 is a level-1 heading and h2 is a level-2 heading. You can create these headings by adding one or more # symbols before your heading text. The number of # you use will determine the size of the heading. 

            

                # This is an h1 tag
