Reference Links

Markdown supports standard reference links and shortcut links

To create a standard reference link you'll need to first define the links using the syntax [link name]: URL

Once the link is defined, you can use it throughout your text by wrapping sections of text with brackets [] and appending a reference to your link [My First Link]


            

                [Google][My First Link]
            


        

            [My First Link]: http://www.google.com

You can use the shortcut syntax like this:

                
This is a link to [Google][].

                
[google]: http://www.google.com "Google"
