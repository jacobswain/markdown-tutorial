Blockquotes

You can show blockquotes with the > character.

This is a blockquote

Markdown supports two types of blockquotes--classic style and lazy style.

With the classic style you include a > character at the beginning of each line.

    
> This is line 1

    
> This is line 2

The lazy style allows you to crate blockquotes by adding a > character at the beginning of the first line.

                
> This is line 1

                
 This is line 2
