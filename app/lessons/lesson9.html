Reference Links

To create a reference link you'll need to first define the links using the syntax [link name]: URL

Once the link is defined, you can use it throughout your text by wrapping sections of text with brackets [] and appending a reference to your link [My First Link]


            

                The Google Self-Driving Car is a project by [Google][My First Link] that involves developing technology
                for autonomous cars.
            


        

            [My First Link]: http://www.google.com
